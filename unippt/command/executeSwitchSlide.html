executeSwitchSlide ​

  • 描述：切换至指定幻灯片 （自然顺序 1 2 3 ...）

  • 语法：unippt.command.executeSwitchSlide(slideIndex:number)

  • 参数：slideIndex: number

  • 示例：

js
// 切换至幻灯片 1
unippt.command.executeSwitchSlide(1);

// 切换至幻灯片 3
unippt.command.executeSwitchSlide(3);
